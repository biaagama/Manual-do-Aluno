# Manual-do-Aluno
O Manual do Aluno SENAI SBO é mais do que um conjunto de regras, ele é um guia que orienta nossa trajetória dentro da escola. Nele, encontramos informações importantes sobre nossos direitos, deveres, normas de convivência, avaliação e postura profissional.
Para mim, o manual representa responsabilidade e preparação para o mercado de trabalho. O SENAI não forma apenas tecnicamente, mas também prepara o aluno para agir com ética, compromisso e respeito. Seguir o manual é entender que disciplina e organização fazem parte do crescimento pessoal e profissional.
No fim das contas, o Manual do Aluno SENAI SBO não é apenas um documento institucional, mas uma base para que cada estudante construa uma trajetória de aprendizado, desenvolvimento e oportunidades.
